隐私规则

1. 我们收集什么信息

Shortlink 是一个注重隐私的短链服务。我们仅收集以下必要信息：

  • 短链数据：您提交的原始链接经过 AES-256-GCM 加密后存储，任何人（包括管理员）无法在未拥有加密密钥的情况下查看原始链接内容。
  • 访问统计：当有人访问您创建的短链时，我们会记录访问的大致地理位置（国家/地区级别）用于统计展示，不记录精确 IP 地址。IP 地址在加密后仅用于速率限制和封禁管理，定期清理。
  • Cookie：管理后台使用必要的 Session Cookie 维持登录状态，不用于追踪或广告目的。

2. 我们不收集什么

  • 不收集个人身份信息（姓名、邮箱、电话号码等）
  • 不在短链页面植入第三方追踪脚本
  • 不出售、分享或交易任何用户数据
  • 不记录用户浏览历史或行为轨迹

3. 数据存储与安全

  • 所有原始链接使用 AES-256-GCM 军事级加密算法存储
  • 加密密钥仅在服务器端保存，不对外暴露
  • 数据库访问受严格权限控制
  • 过期链接自动清理，一次性链接访问后即时禁用

4. 您的权利

  • 您可以随时删除自己创建的短链（通过管理密钥）
  • 您有权知道我们收集了哪些数据
  • 如有隐私相关问题，可通过举报功能联系管理员
